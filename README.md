## 🚀 Problems Covered

### 1. Two Sum
- Approach: Hashing
- Time Complexity: O(n)
- Space Complexity: O(n)

---

### 2. Sort 0s, 1s, 2s
- Approach: Dutch National Flag Algorithm
- Time Complexity: O(n)
- Space Complexity: O(1)

---

### 3. Kadane's Algorithm
- Approach: Maximum Subarray Sum
- Time Complexity: O(n)
- Space Complexity: O(1)

---
### 4. Next Permutation  
Approach: Breakpoint + Swap + Reverse (Optimal Approach)  
Time Complexity: O(n)  
Space Complexity: O(1)  

### 5. Rearrange Array Elements by Sign (Variety 1)  
Approach: Alternate Positive and Negative (Using Extra Space)  
Time Complexity: O(n)  
Space Complexity: O(n)  

### 6. Rearrange Array Elements by Sign (Variety 2)  
Approach: In-place Rearrangement  
Time Complexity: O(n)  
Space Complexity: O(1)  

### 7. Sort 0s, 1s, 2s (Better Approach)  
Approach: Counting Method
